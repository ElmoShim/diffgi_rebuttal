### Overview
- 논문과 supplementary (paper/) submit 했고
- 세 명의 리뷰어에게 리뷰 (review.md) 를 받은 상태임
- rebuttal 초안은 한글로 작성했고 (rebuttal_draft.md)
- 초안 기반으로 실제 latex 로 정리중 (rebuttal/)
- 일단 필요한 내용은 최대한 서술하고 마지막에 분량을 줄이는 작업을 할 것이므로, 따로 분량을 줄이라는 말이 없으면 분량 신경쓰지 않을 것

- reference/ 에 있는 Rebuttal 에서 서술이나 스타일 비슷하게

### RULES
- latex 포멧 관련된 코드는 절대 수정 금지
- rebuttal.tex 의 컨텐츠만 수정 가능 - 나머지 포멧 관련 함수는 수정 금지
- 명시적으로 "수정해줘" 라고 하지 않으면 문서를 수정하지 않을 것
